# K-주전부리 — 6개 부처 제안서 전략 총괄

---

## 사업 핵심 구조

### 컨셉: "전국 인재 × 지역 자산"

```
전국에서 참가자(시니어+청년)를 모집한다.
 → 참가자에게 인구감소지역 5곳의 특산물을 "배정"한다.
 → 지정된 장소(해당 지역이 아닐 수도 있음)에서 교육·프로젝트를 진행한다.
 → 해커톤에서 해당 지역 특산물 기반 시제품·콘텐츠를 완성한다.
 → 카카오메이커스에서 시장검증한다.
 → 해당 인구감소지역의 폐교·폐가를 파일럿 거점으로 전환해 오픈한다.
 → 검증된 모델을 전국 프랜차이즈로 확장한다.
```

### 참가자 ≠ 해당 지역 주민
- 참가자는 전국 어디서든 지원 가능 (해당 지역 주민도 참가 가능하나 한정 아님)
- 배정받은 지역의 특산물로 상품 기획·개발
- 해당 지역 현장 탐방 1~2회 포함 (식재료 발굴·생산자 미팅)

### 교육 장소 ≠ 해당 지역
- 교육·해커톤은 지정 장소에서 진행 (카카오 AI캠퍼스 등)
- 해당 지역 식재료를 교육장으로 조달 (농협 연계)

### 파일럿 거점 = 폐교·폐가 (도시재생)
- 빈점포가 아닌 **폐교·폐가 등 유휴 공간** 활용이 1차 타겟
- 10평 소형이 아닌, 폐교·폐가 규모에 맞는 유동적 설계
- 폐교: 교실→체험공방 / 강당→이벤트 / 운동장→야외마켓 등 복합 활용
- 빈점포 활용은 2단계 (프랜차이즈 전국 확장 시 소형 가맹점)

### 지자체 예산 명분 — "지역 자산 활성화"
참가자가 해당 지역 주민이 아니어도 지자체가 예산을 집행할 수 있는 논리:
1. **지역 특산물 상품화·브랜드화 사업**: 지자체의 특산물이 전국 유통 가능한 상품이 됨
2. **폐교·폐가 활용 도시재생**: 방치된 지역 자산 → 상업·문화 복합 거점으로 전환
3. **지역 브랜드 가치 제고**: "의성 마늘", "보성 녹차" 등 지역명 브랜드 전국 노출
4. **관광·유입 효과**: 거점 오픈 후 외부 방문객 유입 → 생활인구 증가
5. **기존 활용 예**: 지자체가 외부 업체에 위탁해 지역 특산물 축제·상품 개발하는 것과 동일 구조

> 핵심: 돈은 "참가자가 사는 곳"이 아니라 **"결과물이 착지하는 곳"**에서 나온다.

4. 사업 내용 ........................ (5~8p) ★ 가장 두꺼운 섹션
   └ 부처별로 강조하는 프로그램 상세

   └ 4자 협력 R&R + 해당 부처의 역할
