# K-주전부리 — 행정안전부 제안서 기획안

> **제안서 제목 (안)**: 인구감소지역 특산물 상품화 및 폐교·폐가 도시재생 실증 — K-주전부리 활성화 사업 협력 제안

---

## 제안서 설계 방향

행안부는 전체 재원의 65%를 차지하는 핵심 부처다. 제안서의 핀트는 **"인구감소지역의 방치된 자산(특산물·폐교·폐가)을 전국 인재의 역량으로 활성화한다"**는 것이다. 참가자가 해당 지역 주민이 아니어도 지자체가 예산을 집행할 수 있는 논리를 명확하게 세워야 한다. "결과물이 착지하는 곳 = 해당 지자체"이며, 이는 지자체가 외부 전문기관에 축제·상품개발을 위탁하는 것과 동일한 구조다.

## 목차 구성 (총 15~18p)

### 1장. 사업 배경 및 필요성 (3p)

**1-1. 인구감소지역의 방치된 자산**
- 인구감소지역 89곳 지정 현황 (행안부 고시): 국토 59.4%, 인구 9.4%
- 전국 폐교 3,900개+, 인구감소지역 폐가 매년 증가 — 방치된 지역 자산
- 지역 특산물은 높은 품질이지만 상품화·브랜딩·유통 역량 부재
- 소멸기금(지자체당 72억+) 배분 완료되었으나, 효과적 집행 모델 부재

**1-2. 행안부 정책 과제와의 접점**
- 「인구감소지역 지원 특별법」 — 지역 자산 활용, 생활인구 확대
- 「지방소멸대응기금 운용 지침」 — 일자리·상권·유휴 공간 활용 사업
- 「폐교 활용 활성화」 — 폐교의 상업·문화·교육 복합 공간 전환
- 「생활인구 유입」 — 외부 인구의 지역 체류·소비 유도

**1-3. 기존 접근의 한계 → K-주전부리가 다른 이유**
- 기존: 지역 내 인력만 활용 → 인구감소지역은 인력풀 자체가 부족
- K-주전부리: **전국의 인재가 지역 자산을 상품화** → 결과물은 해당 지역에 착지
- 기존: 물리적 환경 개선만 → 콘텐츠·운영 주체 없음 → 재공실화
- K-주전부리: **상품 + 브랜드 + 운영 시스템 + 유통 채널**까지 한 번에 구축
- 기존: 일회성 행사 → 끝나면 원점
- K-주전부리: **프랜차이즈 모델**이 남음 → 지속 가능
- 기존: 개별 지자체 단독 → 비교 불가
- K-주전부리: 5개 지역 동시 실증 → 비교 데이터 확보 → 전국 89곳 확산 근거

### 2장. 사업 목적 (1p)

1. 인구감소지역 5곳의 특산물을 전국 유통 가능한 브랜드 상품으로 상품화
2. 폐교·폐가 등 유휴 공간을 상업·문화 복합 거점으로 전환해 도시재생 실증
3. 소멸기금의 효율적 집행 사례를 만들어 89개 인구감소지역 확산 모델 구축
4. 전국 인재와 지역 자산을 연결하는 새로운 지역 활성화 방법론 제시

### 4장. 사업 내용 — 행안부 관련 핵심 프로그램 (6~8p) ★

**4-1. 전국 모집 → 지역 특산물 배정 구조**
- 시니어 60명 + 청년 20명 전국 공개 모집 (거주지 무관, 역량·의지 기준)
- 팀 구성: 청년 1 + 시니어 3 = 4명 × 20팀
- 20팀을 5개 지역에 4팀씩 배정 (지역당 16명)
- 각 팀은 배정된 지역의 특산물로 상품 기획·개발
- 해당 지역 현장 탐방 1~2회 포함 (식재료 발굴·생산자 미팅)
- 지역 식재료는 농협 연계로 교육장에 조달

**4-2. 교육·프로젝트 (지정 장소)**
- 교육 장소: 카카오 AI캠퍼스, 서울·수도권 교육시설 등
- AI 콘텐츠 제작 교육 6주 + 해커톤 2주
- 배정받은 지역 특산물을 소재로 실전 콘텐츠·시제품 제작

**4-3. 폐교·폐가 거점 전환 — 도시재생**

| 항목 | 내용 |
|------|------|
| 대상 | 인구감소지역 폐교·폐가 등 유휴 공간 5곳 |
| 규모 | 폐교·폐가 규모에 맞게 유동 설계 (교실 단위~건물 단위) |
| 용도 | 지역 특산물 판매 + 체험 + 카페·공방 + 커뮤니티 복합 |
| 인테리어 | K-주전부리 표준 디자인 (규모별 모듈형) |
| 운영 | 지역 위탁 운영 또는 프랜차이즈 가맹 전환 |
| 소유 | 교육청·지자체 재산 → 무상 또는 저가 임대 |

폐교·폐가 활용 이점:
- 복합 용도 가능 (교실→체험공방, 강당→이벤트, 운동장→야외마켓)
- 지역 커뮤니티 기억이 남아 정서적 연결 강함
- 교육청·지자체 소유 → 행정 절차 단순
- 도시재생 사업 명분 명확

**4-4. 지역 특산물 상품화 흐름**
지역 특산물 발굴(농협·지자체) → 레시피 개발(한식진흥원) → 시제품 생산(농협 가공센터) → AI 콘텐츠 제작(교육 산출물) → 카카오메이커스 온라인 검증 → 거점 오프라인 판매 → 프랜차이즈 전국 확장

**4-5. 프랜차이즈 2단계 확장 (2기 이후)**
- 1단계 (파일럿): 폐교·폐가 거점 5곳 = 플래그십/체험형
- 2단계 (확장): 빈점포 소형 가맹점 = 판매 특화
- 2027년 30개 → 2028년 100개+ 전국 가맹
- 폐교 거점 = 체험·관광 / 빈점포 가맹 = 일상 판매로 이원화

### 5장. 추진 체계 (2p)

**5-1. 행안부의 역할 (최소한)**
- 인구감소지역 5곳 지자체에 사업 참여 안내 공문 (일괄 발송)
- 소멸기금 집행 적격 확인 ("지역 자산 활성화" 항목)
- 폐교·폐가 활용 인허가 가이드라인 협조
- 사업 성과 모니터링

> **행안부에 요청하는 것은 "공문 + 적격 확인 + 폐교 가이드"다.** 실행은 3W와 지자체가 담당.

**5-2. 지자체의 역할**
- 소멸기금 내 사업비 편성 (지자체당 약 1.05억)
- 폐교·폐가 무상/저가 제공
- 지역 특산물 생산자·농협 연결
- 인허가 협조

**5-3. 지자체 예산 집행 근거**
> "지역 특산물 상품화 및 유휴 공간(폐교·폐가) 활용 사업"
> → 소멸기금 "지역 경제 활성화" + "유휴 자산 활용" 항목 적격
> → 참가자 거주지와 무관하게, **사업 결과물이 해당 지역에 착지**하므로 집행 가능
> → 지자체가 외부 전문기관에 축제·상품개발을 위탁하는 것과 동일 구조

## 작성 시 유의사항

- **"지역 자산 활성화"가 핵심 프레임**: "주민 참여"가 아닌 "결과물이 착지하는 곳" 논리
- **"추가 예산이 아닌 기 배분 기금"을 반복**: 소멸기금 72억의 1.5%
- **폐교·폐가 도시재생이 차별점**: 빈점포는 2단계. 1단계는 더 상징적인 폐교·폐가
- **전국 모집의 명분**: 인구감소지역은 인력풀 자체가 부족 → 전국 인재를 연결하는 것이 오히려 현실적
- **생활인구 개념 활용**: 참가자의 지역 방문·체류 + 거점 방문객 = 생활인구 정책 부합
- **행안부 역할 최소화**: "공문 + 적격 확인 + 폐교 가이드"이면 이 성과가 행안부 것
