핵심 문제

왜 AI 웹사이트는 다 똑같아 보일까?

🚨 전형적인 AI 웹사이트의 함정

클로드 코드에 "예쁜 웹사이트 만들어줘"라고만 하면, 보라색 그라데이션 + "혁신적인 솔루션" + "AI 자동화" 같은 뻔한 디자인만 나옵니다. 도구의 문제가 아니라 사용 방법의 문제입니다.

❌ 잘못된 접근

클로드 코드에 바로 "웹사이트 만들어줘" → 보라색 그라데이션, AI 냄새 가득한 뻔한 사이트

✅ 올바른 접근

카피 → 디자인 → 코드 순서로 역할 분리 → 7분 만에 차원이 다른 퀄리티

Step 1. 리서치 & 카피 먼저 잡기

코드보다 카피가 먼저다. 문구가 흔들리면 화면도 흔들린다.

  • 1
    타겟 고객 정의 — 클로드에게 이상적인 고객 프로필, 페인포인트, 검색 키워드를 JSON 형태로 추출. JSON은 구조화가 잘 돼 이후 작업에서 더 정확한 결과값이 나옴.
  • 2
    섹션별 카피 생성 — Hero · 문제 제기 · 서비스 소개 · 고객 후기 · CTA 5개 섹션 카피를 한 번에 작성. 결과는 copy.md 마크다운 파일로 저장.
  • 3
    10% 검수 — AI가 90%는 정확히 쓰지만 나머지 10%는 직접 확인 필요. 특히 숫자와 사례가 실제처럼 느껴지는지 체크.
💡 좋은 카피의 원칙

뻔한 말을 구체적인 숫자와 실제 상황으로 바꿔라.
"만족도 높음" → "리포트 만드는 월요일이 사라졌습니다"
"비용 절감" → "도입 4주 만에 운영 비용 40% 절감"
"많은 문의" → "CS 문의 하루 평균 127건"

"카피 15분 안 잡고 들어가면 코드 한 시간을 날려버리게 됩니다."

Step 2. Variant로 디자인 탐색

텍스트만 넣으면 UI 디자인을 자동 생성. 무료 사용 가능.

🛠 Variant란?
🎭

자동 디자인 생성

텍스트(카피)를 붙여 넣으면 여러 가지 UI 디자인을 자동으로 만들어줌. V0와 유사하지만 디자인 탐색에 더 특화.

🔄

다양한 변형 탐색

Strength 슬라이더, 스타일 변경, Remix Color, Shuffle Layout 기능으로 즉시 변형 가능.

🔗

Claude Code로 바로 전달

"Open in" 기능으로 Claude Code, Cursor, Codex 등 다양한 툴로 디자인 코드를 바로 넘길 수 있음.

  • 1
    Hero 섹션 먼저 — Compose 탭에서 copy.md의 히어로 카피를 붙여 넣고 원하는 톤(예: "강렬하게") 요청. 이 첫 섹션이 컬러·폰트·간격·레이아웃 등 사이트 전체 톤을 결정.
  • 2
    "New chat from design"으로 연결 — 마음에 드는 디자인의 ⋯ 버튼 → "New chat from design" 클릭. 이 디자인 시스템을 기반으로 다음 섹션 생성.
  • 3
    섹션마다 카피 직접 붙여넣기 — "이 디자인 시스템을 따라서 아래 카피로 [섹션명]을 만들어줘" + copy.md 내용 붙여넣기. 카피를 직접 넣어야 같은 톤이 유지됨.
  • 4
    5개 섹션 완성 — Hero → 문제 제기 → 서비스 → 고객 후기 → CTA 순서로 동일한 방식 반복. 실제로는 5분 이내 완성 가능.
⚠️ 반드시 기억할 것

카피를 같이 넘기지 않으면 AI가 카피를 멋대로 만들어버려 처음 잡은 방향이 무너집니다.
섹션마다 따로 시키면 페이지가 조각납니다. 반드시 이전 디자인을 참조로 연결하세요.

핵심 원칙

이 두 가지만 기억하세요

01

카피부터 정리하라

카피 15분 안 잡고 들어가면 코드 한 시간을 날린다. 문구가 흔들리면 화면도 흔들린다. 카피 파일이 있어야 디자인이 고정된다.

02

디자인 톤은 Hero 하나에서 잠궈라

폰트·색·밀도를 히어로 섹션에서 결정하면, 나머지는 그걸 따라가면 된다. 섹션마다 따로 시키면 페이지가 조각난다.
