# K-주전부리 — 중소벤처기업부 제안서 기획안

> **제안서 제목 (안)**: 지역 특산물 기반 소자본 프랜차이즈 창업 실증 — K-주전부리 활성화 사업 협력 제안

---

## 제안서 설계 방향

중기부에는 **"해커톤에서 검증된 레시피를 전국 누구나 소자본으로 창업할 수 있는 프랜차이즈로 만드는 프로젝트"**로 설명한다. 교육에서 끝나는 게 아니라 실제 점포가 열리고, 매출이 발생하고, 전국 가맹으로 확장되는 흐름을 보여줘야 한다. 예산 요청은 기존 프로그램(예비창업자 패키지, 소공인 특화자금)에 연결하는 구조.

---

## 목차 구성 (총 13~15p)

### 1장. 사업 배경 및 필요성 (2p)

**1-1. 인구감소지역 소상공인의 현실**
- 인구감소지역 89곳의 상업 공간 공실률 심화
- 지역 특산물 = 높은 품질이지만 상품화·유통·브랜딩 역량 부재
- 청년 창업가의 지역 연계 진입 장벽: 정보·네트워크·초기 비용 부재
- 기존 창업 지원: 교육 수료 후 후속 없음 → 실제 창업 전환율 저조

**1-2. 중기부 정책 과제와의 접점**
- 「소상공인 기본계획」 — 로컬 브랜드 육성, 소자본 창업 모델 개발
- 「예비창업자 지원사업」 — 실전형 창업 프로그램, 활동비 지원
- 「소공인 특화자금」 — 소규모 제조업 자금 지원
- 「전통시장·상권 활성화」 — 빈점포 활용, 상권 재생

**1-3. K-주전부리가 다른 이유**
- 기존 창업 지원: 아이디어 → 교육 → (탈락) → 혼자 창업 시도
- K-주전부리: 아이디어 → 교육 → 해커톤(시제품 완성) → 카카오메이커스(시장검증) → 파일럿 거점(실증) → **프랜차이즈(전국 확장)**
- 참여팀 자부담 0원. 시제품 생산비는 소공인 특화자금 융자, 매출로 상환
- 전국에서 예비창업자를 모집하여 인구감소지역 특산물과 연결

---

### 2장. 사업 목적 (1p)

1. 전국 공모 청년 예비창업자 20명을 지역 특산물 기반 실전 창업으로 연결
2. 해커톤 결과물(레시피 20종)을 표준화해 소자본 프랜차이즈 모델 실증
3. 인구감소지역 5곳에 폐교·폐가 파일럿 거점을 오픈해 운영 데이터 확보
4. 카카오메이커스를 통한 온라인 시장검증으로 수요 기반 창업 모델 제시

---

### 3장. 사업 개요 (2p)

| 항목 | 내용 |
|------|------|
| 사업명 | K-주전부리 활성화 사업 |
| 참여 규모 | 시니어 60명 + 청년 20명 = 80명, 20팀 (**전국 공개 모집**) |
| 참가 방식 | 참가자에게 5개 지역 특산물 배정 → 지정 장소에서 교육·프로젝트 |
| 교육 장소 | 카카오 AI캠퍼스 등 (해당 지역 한정 아님) |
| 파일럿 거점 | 폐교·폐가 등 유휴 공간 전환 5곳 |
| 컨소시엄 | 지방시대위(MOU 완료) · 카카오 · 숙명여대 · 농협 · 농민신문 |

**12주 흐름 — "창업 파이프라인" 관점**
> 예비창업자 전국 모집 → 지역 특산물 배정 → AI 교육(창업 역량 강화) → 해커톤(시제품·브랜드 완성) → 메이커스 펀딩(온라인 시장검증) → 파일럿 거점(오프라인 실증) → 프랜차이즈 전환(전국 확장)

---

### 4장. 사업 내용 — 창업 실증 핵심 프로그램 (5~6p) ★

**4-1. 제품 개발 — 해커톤에서 레시피가 탄생하는 과정**

| 단계 | 내용 | 지원 주체 |
|------|------|---------|
| 지역 식재료 발굴 | 5개 인구감소지역 특산물 선정·확보 (교육장 조달) | 농협·지자체 |
| 레시피 개발 | 식품 전문가 컨설팅 + 팀별 시제품 기획 | 한식진흥원 (무상) |
| 해커톤 고도화 | 카카오 AI캠퍼스에서 2일간 시제품+콘텐츠 동시 완성 | 카카오 (공간) |
| 식품안전 검사 | 20종 위생·안전 검사 | 지자체 보건소 |
| 소량 생산 | 20종 × 500개 | 농협 가공센터 + 소공인 특화자금 |
| 패키징 | 표준 디자인 20종 | KCDF 전문가 매칭 (무상) |

**4-2. 시장검증 — 카카오메이커스 펀딩**
- 해커톤 결과물 20팀 전량 카카오메이커스 론칭
- AI 자동생성 상세페이지 + 참여팀 콘텐츠 → 실제 소비 반응 확인
- 펀딩 달성률·주문 수·재구매율 → 수요 데이터 기반 거점 오픈 결정
- 우수 제품은 "카카오한입" 브랜드로 카카오메이커스 단독 론칭

**4-3. 파일럿 거점 — 폐교·폐가 복합 공간**

| 항목 | 내용 |
|------|------|
| 대상 | 인구감소지역 폐교·폐가 등 유휴 공간 5곳 |
| 규모 | 폐교·폐가 규모에 맞게 유동 설계 (교실~건물 단위) |
| 용도 | 지역 특산물 판매 + 체험 + 카페·공방 + 커뮤니티 복합 |
| 인테리어 | K-주전부리 표준 디자인 (규모별 모듈형) |
| 원재료 공급 | 농협 연계 지정 공급사 규격 식자재 |
| 운영 시스템 | WorkB POS 기반 매출·재고 통합 관리 |

**4-4. K-주전부리 프랜차이즈 2단계 확장 로드맵**

| 단계 | 시기 | 내용 | 규모 |
|------|------|------|------|
| 1단계 — 파일럿 | 2026.11~12 | 폐교·폐가 거점 5곳 운영. 체험+판매 플래그십 | 5개 거점 |
| 2단계 — 1차 가맹 | 2027 상반기 | 빈점포 소형 가맹점 모집. 파일럿 데이터 기반 | 20~30개 |
| 3단계 — 공급망 | 2027 하반기 | 농협 가공공장 가동. 규모의 경제 | 50개 |
| 4단계 — 전국 확장 | 2028~ | WorkB POS 기반 통합 관리 | 100개+ |

- 1단계(폐교·폐가) = 플래그십/체험형 거점
- 2단계 이후(빈점포) = 소형 판매 가맹점으로 이원화

**4-5. 창업 비용 역산 — 소자본 가능성 (2단계 빈점포 가맹 기준)**

| 항목 | 일반 창업 | K-주전부리 창업 |
|------|---------|-------------|
| 인테리어 | 5,000~8,000만 | 3,000만 (표준 패키지) |
| 설비 | 2,000~3,000만 | 800만 (표준 설비) |
| 초기 재료비 | 500~1,000만 | 500만 (농협 연계) |
| 가맹비 | 1,000~3,000만 | 500만 (추정) |
| 임대보증금 | 2,000~5,000만 | 0원 (빈점포 무상 or 저가) |
| **합계** | **1~2억** | **약 4,800만** |
| 정부 지원 적용 시 | — | **약 2,000만 이하** |

**4-6. 참여팀 자부담 0원 구조**
- 교육비: 문체부 (강사비) + 숙명여대 (커리큘럼)
- 활동비: 지자체 소멸기금(시니어) + 중기부 예비창업자 패키지(청년)
- 시제품 생산: 농협 원재료(현물) + 소공인 특화자금(융자, 매출 상환)
- 패키징: KCDF 무상 매칭
- 판매 채널: 카카오메이커스(무료) + 거점 점포(폐교·폐가 무상)

### 5장. 추진 체계 (1~2p)

**중기부의 역할**
- 예비창업자 패키지에 청년 창업가 20명 연결
- 소공인 특화자금 무이자 융자 연계 (시제품 생산)
- 소상공인 컨설팅 무상 제공 (법률·세무·노무)
- 프랜차이즈 가맹 등록 컨설팅 (소상공인시장진흥공단)

3. 프랜차이즈 가맹 등록 컨설팅 연결

## 작성 시 유의사항

- **프랜차이즈 2단계가 핵심**: 1단계 폐교(플래그십) → 2단계 빈점포(소형 확장)
- **창업 비용 역산 테이블 필수**: "4,800만 원, 정부 지원 시 2,000만 이하"는 매력적
- **전국 모집 프레임**: "어디서든 지원 가능한 예비창업자"
- **카카오메이커스 시장검증**: "수요가 확인된 후 거점을 연다" = 리스크 최소화
- **기존 프로그램 연계**: 새 예산이 아니라 "이미 있는 프로그램에 우리 참여자를 등록"
