# K-주전부리 — 국토교통부 제안서 기획안

> **제안서 제목 (안)**: 인구감소지역 폐교·폐가 전환형 도시재생 실증 — K-주전부리 활성화 사업 협력 제안

---

## 제안서 설계 방향

국토부에는 직접 예산을 요청하지 않는다. **폐교·폐가를 커머스·문화·커뮤니티 복합 거점으로 전환하는 도시재생 실증**으로 제안서를 구성하고, 기존 도시재생 뉴딜 사업과의 연계를 제안한다. 1기에서는 관계 구축 + 레퍼런스 확보가 목표, 2기 이후 국토부 예산 본격 연계.

---

## 목차 구성 (총 10~12p)

### 1장. 사업 배경 및 필요성 (2p)

**1-1. 인구감소지역 공간 공동화**
- 전국 폐교 3,900개+, 인구감소지역 폐가 급증 — 지역 자산 방치
- 폐교·폐가 방치 → 경관 훼손 → 안전 위험 → 추가 인구 유출 악순환
- 기존 빈집정비 사업: 주거 중심, 폐교·상업 공간은 후순위

**1-2. 국토부 정책 과제와의 접점**
- 「도시재생 뉴딜 로드맵」 — 일반근린형 도시재생, 상권 활성화
- 「빈집정비 사업」 → 폐교·폐가까지 확장 필요성
- 「생활SOC 복합화」 — 문화·커머스·커뮤니티 복합 공간
- 「스마트시티 챌린지」 — 상권 데이터 기반 지역 관리

**1-3. K-주전부리가 다른 이유**
- 기존 도시재생: 물리적 환경 개선(인테리어·파사드) 위주 → 콘텐츠·운영 주체 없음 → 재공실화
- K-주전부리: 폐교·폐가 + 운영 주체(전국 모집 팀) + 상품(해커톤 레시피) + 유통(카카오) + 관리(WorkB POS) = **"하드웨어 + 소프트웨어 + 운영" 통합 모델**
- "열면 손님이 온다"가 아니라 **"팔 상품과 유통 채널이 있는 상태로 연다"**

---

### 2장. 사업 목적 (1p)

1. 인구감소지역 폐교·폐가 5곳을 상업·문화 복합 거점으로 전환 실증
2. 물리적 환경 개선 + 운영 콘텐츠 + 디지털 관리를 결합한 통합 도시재생 모델 제시
3. 파일럿 데이터를 기반으로 89곳 확산 가능성 검증

---

### 3장. 사업 개요 (2p)

| 항목 | 내용 |
|------|------|
| 사업명 | K-주전부리 활성화 사업 |
| 참여 규모 | 시니어 60명 + 청년 20명 = 80명, 20팀 (전국 모집) |
| 파일럿 거점 | 인구감소지역 폐교·폐가 등 유휴 공간 5곳 |
| 컨소시엄 | 지방시대위(MOU 완료) · 카카오 · 숙명여대 · 농협 · 농민신문 |

**12주 흐름 — "공간 전환" 관점**
> 지역 선정(폐교·폐가 확보) → 전국 참가자 모집(특산물 배정) → 교육·해커톤(거점 콘텐츠 개발) → 인테리어·설비(모듈형 공간 조성) → 거점 오픈(운영 시작) → 데이터 수집(상권 파급 효과) → 프랜차이즈(전국 확산)

---

### 4장. 사업 내용 — 폐교·폐가 전환 프로그램 (4~5p) ★

**4-1. 폐교·폐가 전환 과정 상세**

| 단계 | 내용 | 기간 |
|------|------|------|
| 대상 발굴 | 지자체 유휴 재산 목록 + 교육청 폐교 목록 조사 | 04~05월 |
| 소유자 협의 | 교육청(폐교) / 지자체(폐가) 무상임대 또는 매입 검토 | 05~06월 |
| 용도 확인 | 건축법상 용도전환 가능 여부 확인 | 06월 |
| 모듈형 설계 | 공간 규모에 맞는 K-주전부리 표준 디자인 적용 | 07~08월 |
| 인테리어 시공 | 전국 거점 통일 패키지 (규모별 모듈) | 09~10월 |
| WorkB POS 설치 | 매출·재고 디지털 관리 시스템 | 10월 |
| 거점 오픈 | 운영 시작 | 11월 |

**4-2. 폐교 공간 활용 설계**

| 기존 공간 | 전환 용도 | 비고 |
|---------|---------|------|
| 교실 (1~2칸) | 체험 공방 — 주전부리 만들기 체험 | 관광객·방문객 대상 |
| 교실 (1칸) | 판매 공간 — 지역 특산물 상품 진열·판매 | 카카오 POS 연동 |
| 급식실/가정실 | 카페·간식 조리 공간 | 즉석 조리 메뉴 |
| 강당 | 이벤트·워크숍 공간 | 지역 커뮤니티 활용 |
| 운동장 | 야외 마켓·시즌 행사장 | 주말 마켓 운영 가능 |

**4-3. 기존 도시재생과의 차이**

| 기존 도시재생 | K-주전부리 도시재생 |
|-----------|------------|
| 물리적 환경만 개선 | 공간 + **상품 + 유통 + 운영** 통합 |
| 운영 주체 불명확 | 프랜차이즈 모델로 운영 체계 확정 |
| 일회성 보조금 | 매출 기반 자생 구조 |
| 개별 지자체 단독 | 5곳 동시 실증 → 비교 데이터 |
| 성과 측정 어려움 | WorkB POS 데이터 자동 수집 |

**4-4. 상권 파급 효과 측정 설계**
- 거점 반경 500m 유동인구 변화 (카카오맵 데이터 활용)
- 거점 일 매출·고객 수 (WorkB POS 자동 수집)
- 거점 방문객 출발지 분석 (생활인구 데이터)
- 지역 주민 인식 변화 (사전·사후 설문)
- 6개월 운영 데이터 → 도시재생 연계 효과 보고서 작성

**4-5. 프랜차이즈 확장과 도시재생의 연결**
- 1단계(폐교·폐가) = 플래그십/체험형 거점 → 관광·생활인구 유입
- 2단계(빈점포) = 소형 판매 가맹점 → 일상 상권 활성화
- 전국 가맹점 = 전국 도시재생 거점 네트워크
- 89개 인구감소지역에 동일 모델 적용 가능 = 정책 성과

### 5장. 추진 체계 (1p)

**국토부의 역할 (0원)**
- 도시재생 뉴딜 사업지 내 K-주전부리 거점 배치 검토
- 폐교·폐가 용도변경(건축법) 인허가 간소화 협조
- 1기 성과 발표 시 도시재생 우수 사례 등록 검토
- 2기 확장 시 신규 도시재생 사업지 연계 검토

---

### 6장. 기대 효과 및 성과 지표 (1~2p)

| 지표 | 목표치 |
|------|-------|
| 폐교·폐가 전환 | 5곳 → 상업·문화 복합 거점 |
| 상권 파급 데이터 | 6개월 운영 (유동인구·매출·방문객 출발지) |
| 도시재생 연계 실증 | 1건 (민관학 협력 모델) |
| 표준 거점 모델 | 1종 (폐교형 + 폐가형 모듈 확정) |
| 전국 확산 가능성 | 89개 인구감소지역 적용 가능 |

### 8장. 협조 요청 (1p)

1. 국토부 도시재생과 실무 협의 (지방시대위 경유)
2. 5개 지역 도시재생 사업 연계 가능성 확인
3. 폐교·폐가 용도변경 인허가 간소화 가이드라인 검토
4. 1기 성과 발표 시 도시재생 우수 사례 등재

## 작성 시 유의사항

- **폐교·폐가 Before/After가 핵심 비주얼**: 컨셉 이미지 반드시 포함
- **"비용 0에 성과 확보"를 명확히**: 국토부 입장에서 부담 없는 제안
- **"기존 사업에 붙이는" 프레임**: 신규 사업이 아닌 기존 도시재생 연계
- **상권 파급 효과 측정 설계를 구체적으로**: 국토부는 데이터를 중시
- **1기는 관계 구축, 2기부터 본격 연계**: "장기 파트너십 제안" 톤
- **전국 모집 참가자가 지역을 방문**: 현장 탐방 = 생활인구 유입 데이터
