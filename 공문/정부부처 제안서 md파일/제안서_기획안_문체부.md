# K-주전부리 — 문화체육관광부 제안서 기획안

> **제안서 제목 (안)**: AI 기반 시니어 크리에이터 양성 및 지역문화 거점 조성 사업 — K-주전부리 활성화 사업 협력 제안

---

## 제안서 설계 방향

문체부에는 "K-콘텐츠 교육 + 시니어 문화인력 양성 + 폐교 문화거점화 + 공공 디지털 자산 기부체납"이라는 4중 접점이 있다. 교육 커리큘럼과 콘텐츠 산출물, 기부체납 구조가 충분히 설명된 후에 예산 이야기로 넘어가야 한다. "이건 우리 부처 콘텐츠 정책 성과로 올릴 수 있겠다"가 목표.

---

## 목차 구성 (총 15~18p)

### 1장. 사업 배경 및 필요성 (2~3p)

**1-1. 세 가지 문화 공백**
- 시니어 디지털 문화 격차: 50대+ AI 도구 활용 역량 전무. 기존 문화 교육은 수동적 "감상" 중심
- 지역 특산물 콘텐츠 부재: 인구감소지역 특산물은 품질 우수하나 상세페이지·숏폼·브랜드 영상 등 판매 접점 콘텐츠가 없어 전국 유통 불가
- 폐교 3,900개+ 방치: 지역의 문화적 기억이 담긴 공간이 사라지고 있음 → 문화거점 전환 가능성

**1-2. 문체부 정책 과제와의 접점**
- 「지역문화진흥기본계획」 — 지역 문화인력 양성, 생활문화 활성화
- 「콘텐츠산업 진흥계획」 — AI 활용 콘텐츠 제작 역량 확산
- 「시니어 문화활동 지원」 — 시니어 세대 문화 참여 확대
- K-콘텐츠 해외 확산 — 지역 먹거리·문화를 한류 콘텐츠로 연결
- 폐교 문화재생 — 폐교를 문화·예술·커뮤니티 공간으로 전환

**1-3. K-주전부리가 다른 이유**
- 기존 시니어 문화 교육: 교육으로 끝남 → 실전 활용 없음
- K-주전부리: **교육 → 콘텐츠 제작 → 시장검증 → 거점 오픈 → 프랜차이즈** 연결
- "배운 걸로 실제 상품을 만들어 카카오에서 판다" = 실전형 문화 교육의 최종 형태

---

### 2장. 사업 목적 (1p)

1. 전국에서 모집한 시니어 60명을 AI 활용 콘텐츠 크리에이터로 양성
2. 인구감소지역 5곳 특산물의 콘텐츠화 (상세페이지·숏폼·브랜드 영상)
3. 폐교·폐가를 K-주전부리 문화·커머스 복합 거점으로 전환
4. AI 콘텐츠 교육 커리큘럼을 개발해 전국 문화시설 보급 모델 실증
5. 베테랑 앱(시니어 매칭 플랫폼)을 공공 디지털 자산으로 기부체납

| 항목 | 내용 |
|------|------|
| 사업명 | K-주전부리 활성화 사업 |

| 참가 방식 | 참가자에게 지역 특산물 배정 → 지정 장소에서 교육·프로젝트 |
| 교육 장소 | 카카오 AI캠퍼스 등 지정 장소 (해당 지역 한정 아님) |

> 모집 → 특산물 배정 → **AI 교육(1~6주)** ← 강사비·커리큘럼 → **해커톤(7~8주)** → **콘텐츠 고도화(9~10주)** ← 영상 제작비 → 거점 오픈(11주) → 성과 발표(12주)

### 4장. 사업 내용 — 문체부 관련 핵심 프로그램 (5~7p) ★

**4-1. AI 콘텐츠 제작 교육 프로그램**

교육 대상: 시니어 60명 (50대 이상, 전국 모집)
교육 기간: 6주 집중 + 해커톤 2주 실전 적용
교육 장소: 지정 장소 (카카오 AI캠퍼스 또는 수도권 교육시설)

| 주차 | 교육 내용 | AI 도구 | 산출물 |
|------|---------|--------|--------|
| 1~2주 | AI 기초 리터러시 + 이미지 생성 | Midjourney, DALL-E | 제품 컨셉 이미지 |
| 3주 | 판매 카피·스토리텔링 작성 | ChatGPT | 브랜드 스토리, 판매 문구 |
| 4주 | 상세페이지 디자인·구성 | ChatGPT + Midjourney | 상세페이지 초안 |
| 5~6주 | 숏폼 영상 기획·편집 | CapCut, Vrew | 숏폼 영상 1인 1편 |
| 7~8주 | 해커톤 — 시제품·콘텐츠 동시 완성 | 전 도구 통합 | 팀별 완성 콘텐츠 패키지 |

- 강사: AI 전문 강사 3명 + 숙명여대 교수진
- 교육 방법론: "배운다 < 만든다" — 매 주차 실제 판매 접점 콘텐츠 산출
- 배정받은 지역 특산물을 소재로 실전 콘텐츠 제작 (식재료는 농협 연계 교육장 조달)
- 커리큘럼 개발: 3W + 숙명여대 공동. 교육 종료 후 표준 커리큘럼으로 정리 → 전국 보급 제안

**4-2. 콘텐츠 산출물 총괄**

| 산출물 | 수량 | 용도 | 제작 주체 |
|--------|------|------|---------|
| 상세페이지 | 20종 | 카카오메이커스 온라인 판매 | AI 자동생성 + 참여팀 편집 |
| 숏폼 영상 | 40편 | 인스타·틱톡 홍보 | 참여자 제작 + 편집자 보정 |
| 브랜드 영상 | 1편 (메인) + 20편 (팀별) | K-주전부리 브랜드 홍보 | 영상 PD + 참여자 공동 |
| 브랜드 스토리 | 20종 | 패키지·펀딩 페이지 | 참여팀 AI 활용 |
| 교육 기록 영상 | 10편 | 커리큘럼 보급용 | 영상 PD 촬영 |

**4-3. 폐교 문화거점화**
- 폐교를 K-주전부리 체험·판매·커뮤니티 복합 공간으로 전환
- 교실 → 체험 공방 / 강당 → 이벤트 공간 / 운동장 → 야외 마켓 등
- K-푸드 체험 + AI 제작 스토리 연계 → 관광 콘텐츠로 확장 가능성
- 2기 이후: 한류 콘텐츠 확장 (외국인 관광 접점)

**4-4. 베테랑 앱 기부체납 구조**

| 항목 | 내용 |
|------|------|
| 앱 명칭 | 베테랑 — 시니어-청년 소셜 믹스 매칭 플랫폼 |
| 주요 기능 | 시니어 인력 DB, 프로젝트 매칭 알고리즘, 운영 데이터 대시보드 |
| 개발 방식 | 바이브코딩 활용, WorkB 모듈 재활용, 3개월 MVP |
| 개발 비용 | 총 4,000만 (3W 자체 2,500만 + 문체부 위탁 1,500만) |
| 기부체납 시기 | 사업 종료 후 (2026.12~2027.01) |
| 기부체납 대상 | 문화체육관광부 (또는 산하 기관) |
| 이전 범위 | 앱 소스코드 + 운영 데이터 + 매칭 알고리즘 + 운영 매뉴얼 |
| 절감 효과 | 정부 독자 구축 시 수억~수십억 추정 → 민간 개발 후 기부체납 |

**4-5. 세대융합 팀 구성**
- 1팀 = 청년 창업가 1명 + 시니어 3명 (개발자 + 디자이너 + 마케터)
- 시니어 3가지 전문 역할: 프론트엔드 개발자 / 디자이너 / 콘텐츠 마케터
- AI 도구가 시니어의 "경험적 판단력"을 "실행 가능한 결과물"로 변환하는 구조

## 작성 시 유의사항

- **"기부체납"이 최대 무기**: 0.9억 투입으로 공공 디지털 자산 확보. 제안서 4장 후반에 배치.
- **콘텐츠 산출물 숫자 구체적으로**: 270건+ = 문체부 담당자 보고에 바로 쓸 수 있는 수준
- **교육 장소가 지역이 아닌 점 자연스럽게**: "집중 교육의 효율성 + 현장 탐방 병행" 프레임
- **전국 모집이 오히려 강점**: "전국 시니어의 디지털 문화 격차 해소" = 지역 한정보다 정책 범위 넓음
- **폐교 = 빈점포보다 문체부에 매력적**: 문화재생 명분 명확
- **한류·관광 연계는 2기 비전으로 가볍게**: 1기에서 과도한 약속 금지
- **위탁용역 계약은 별도 문서**: 제안서에는 구조만 설명
