# K-주전부리 회의 메모 — 미결 사항 및 고민 정리

---

## 1. 핵심 목적 재확인

**주목적:** 지역 활성화 · 세대간 일자리 창출 · 도시재생

### 일자리 창출 논리 연결 (가장 큰 의문점)

- 시니어(40·50대 이상)는 AI 역량이 낮음 → 교육 자체에 의미
- AI 교육 + 브랜딩·패키징 실무 경험 → 차후 취업 연결 고리
- **문제:** 창업지원 ↔ 채용은 방향이 정반대 → 논리 연결 필요

**해결 방향 (고용부 연계):**
- 고용부가 공기업·사기업과 특약 체결 → 수료자 면접 기회 보장
- 고용부가 기업에 직접 시니어 채용 요청 + 혜택 제공
- 베테랑 앱 마이데이터 연동 → 일자리 파이프라인 구성

**핵심 연결고리:** `시니어 일자리 창출 ← AI·실무 교육 → 고용부 파이프라인`

## 2. 운영 구조 미결 사항

### 팀 구성
| 구분 | 인원 | 역할 |
|------|------|------|
| 시니어 | 2~3명 (40대 이상) | 브랜딩, 패키징, 마케팅, 디자인, 기획 |
| 주니어 | 1명 | 요식업 종사자 또는 관심자 (아이디어 발굴) |

- 시니어 소싱: 3W (베테랑 앱 등 활용)
- 주니어 소싱: 카카오 (미협의 — 협의 필요)
- 선정 방식: 섭외(멘토·멘티 형식) + 공개 모집 병행

### 오프라인 장소 (미결)
- 교육·해커톤 거점: **카카오 AI캠퍼스** (총괄 기획안 기준) — 용인캠퍼스 여부 재확인 필요
- 식재료는 해당 지역에서 조달 (농협 연계)
- 현장 탐방: 배정 지역 1~2회

### 전국 운영 구조 (미결)
- 도시별 선정: 수도권·부산·대구·광주·대전 등
- 전국 지사 운영 주체 → 미정
- 전국 강사 수급: 고용부 또는 문체부 연계 검토

### 팀 이탈 방지 대책 (미결)
- 팀원 이탈·갈등·형평성 문제 발생 시 대응 방안 없음
- 지분·수익 분배 규정 필요 (레시피 공유 범위 포함)

## 3. 12주 프로그램 타임라인

```
1~4주   실교육 (AI, 브랜딩, 패키징, 마케팅)
5~10주  시제품 개발 + 마케팅·패키징 작업
11~12주 카카오메이커스 출시 + 펀딩 진행
        → 수상 기준: 판매량, 상품성, 펀딩 달성률
9월     수상식
```

> 3달(12주)이 길다는 의견 있음 — 단축 방안 검토 필요

## 4. 사업 구조 (확정된 부분)

1. **지역 활성화**: 인구감소지역 특산물 기반 디저트 상품화 → 전세계 유통 목표
2. **해커톤**: 시니어 + 주니어 팀으로 지역 특산물 시제품 기획·개발
3. **카카오메이커스**: 온라인 독점 판매 + 펀딩 병행 / 포장·배송·정산 지원
4. **도시재생**: 폐교·폐가 활용 → 체험공방·소형 거점 전환 (팝업스토어 등)

## 5. 부처별 역할 요약 (고민 포인트)

| 부처 | 역할 | 고민 |
|------|------|------|
| **행안부** | 소멸기금 + 지자체 조율, 특산물·소멸 위기 지역 선정 | 예산 가장 큼, 지자체 연계 선행 필요 |
| **고용부** | 시니어·주니어 일자리 창출, 회사 소싱, 면접 파이프라인 | 베테랑 앱 마이데이터 연동 검토 |
| **중기부** | 팀 사후지원 (융자·창업지원) | 역할 작음, 기존 프로그램 연결 위주 |
| **문체부** | AI 교육·콘텐츠, K-문화 홍보, 베테랑 앱 기부체납 | 기부체납 후 유지보수 운영계약 구조화 필요 |
| **국토부** | 폐교·폐가 장소 지원 | 행안부와 연계 움직여야 함 |
| **방통위** | 영상·미디어 관련 지원 | OTT 진출은 현실성 낮음, 미디어센터 현물 위주 |
